WebAuthn Demo

Web Authentication API (WebAuthn) enables passwordless/usernameless authentication through your FIDO compliant Security Key.

Register your FIDO device using the 'Register device' button below, confirm by pressing the button on KeyVault and click 'Allow' when asked whether the site can see your security key. You can then authenticate using the registered credential.

Note: MacOS currently supports WebAuthn standard only via USB, while iOS via NFC and USB. Windows and Android users can use USB, BLE and NFC. Linux users please start with USB first.
